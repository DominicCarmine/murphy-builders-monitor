🔴 Ceramic Products
+12.0% over 3 years

Zero quarters of decline across 3 years. Most consistent escalator in the dataset. Apply 4.0–5.0% p.a. Trades: tiling, bathrooms, facade, floor finishes.

🔴 Cement Products
+11.3% over 3 years

Accelerated in 2025 with +8 points in 12 months. Apply 3.5–4.5% p.a. Trades: blockwork, masonry, precast, tilt panels.

🟠 Timber, Board & Joinery
+5.5% over 3 years

Re-escalating after 2023 dip. Heavy index weighting amplifies impact. Apply 2.5–3.5% p.a. Trades: framing, formwork, joinery, fitout, flooring.

🟠 Electrical Equipment
+6.8% over 3 years

High volatility — quarterly swings up to 5 points. Apply 2.0–3.0% p.a. Consider early procurement or R&F clauses for projects >12 months.

🟡 Concrete, Cement & Sand
+7.5% over 3 years

Slow but persistent — no quarters of decline. Apply 2.0–2.5% p.a. Trades: structural concrete, slabs, civil works, foundations.

🟡 Other Metal Products
+4.7% over 3 years

Largest index weighting (29 contribution points). Steady low-volatility escalation. Apply 1.5–2.0% p.a. Trades: structural connections, cladding, framing, balustrades.

🟢 Steel Products
−10.7% over 3 years

Only category showing real deflation. Peaked at 163 (2022 Q1), now 125. Approaching floor — model flat. Treat further deflation as upside only.

🟢 Plumbing Products
−2.6% over 3 years

Peaked mid-2023, gently deflating. Apply 0–1.0% p.a. Monitor for reversal if approvals-driven demand increases.

How to Set Up Automated Email Alerts

To receive automated email notifications when thresholds are breached, ask your IT team to:

  • Set up a monthly scheduled task (Windows Task Scheduler or cron job) to run a simple script
  • The script should fetch the ABS API endpoints listed in the About page
  • Compare fetched values against the thresholds configured above
  • Send an email via your company SMTP server if any threshold is breached
  • ABS API endpoints: api.data.abs.gov.au/data/BA/1.0/AUS — Building Approvals
  • ABS API endpoints: api.data.abs.gov.au/data/HF/1.0/AUS — Housing Finance

ℹ️ About This Dashboard
Methodology, data sources, and guidance on how to use this tool effectively.

📊 What This Dashboard Does

This tool monitors key economic indicators that lead construction cost movements by 6–18 months, giving Murphy Builders estimators advance warning of cost escalation before it appears in supplier and subcontractor pricing. Version 2.0 adds construction loan commitments, construction commencements, pipeline indicator framework, project calculator, tender tracker, and alert thresholds.

🔗 The Lead-Lag Relationship Explained

The dashboard is built around a well-established economic chain:

  • Step 1: Banks approve construction loans → developer pipeline confirmed (18 mth lead)
  • Step 2: Developers purchase land → PEXA settlements rise (15 mth lead)
  • Step 3: DAs lodged → planning pipeline growing (12 mth lead)
  • Step 4: Building approvals granted → construction imminent (9 mth lead)
  • Step 5: Commencements rise → immediate demand pressure (3–6 mth lead)
  • Step 6: PPI rises → cost escalation confirmed (lagging)
  • Step 7: Tender prices increase → impact on Murphy Builders

🔄 Data Update Schedule

  • Building Approvals (ABS 8731.0): Monthly — ~5 weeks after reference month
  • Housing Finance (ABS 5601.0): Monthly — ~5 weeks after reference month
  • Construction Commencements (ABS 8752.0): Quarterly — ~10 weeks after quarter end
  • Producer Price Index (ABS 6427.0): Quarterly — ~4 weeks after quarter end
  • PEXA Market Insights: Monthly — manually entered from pexa.com.au/market-insights
  • DA Lodgements: Monthly — manually entered from state planning portals

📡 ABS API Information

This dashboard connects to the ABS SDMX-JSON API at api.data.abs.gov.au. The API is free, requires no authentication, and is updated automatically when ABS publishes new data. If live data cannot be retrieved (e.g. network restrictions), the dashboard falls back to the embedded historical dataset which is accurate to December 2025.

⚠️ Important Notes for Estimators

1. This tool informs — it does not replace judgement. Always validate against current subcontractor and supplier feedback.

2. Lead indicators show direction, not exact magnitude. Use PPI trend data for magnitude guidance.

3. Review escalation allowances quarterly minimum. Market conditions shift rapidly.

4. For projects with programs exceeding 12–18 months, strongly consider rise and fall provisions tied to ABS PPI indexes.

5. The Tender Tracker is your most valuable calibration tool — the more records entered, the more accurately it will reflect Murphy Builders' specific project experience.
